# LOOP
| Loop Type  | Use Case Example                      |
| ---------- | ------------------------------------- |
| `for`      | Jab repeat count **pata ho**          |
| `while`    | Jab repeat count **na pata ho**       |
| `do-while` | Jab at least **ek baar run** karna ho |


##  while loop

```java
while (condition){
    // Do something
}

```
"Jab tak condition true hai, tab tak repeat karte raho!"
